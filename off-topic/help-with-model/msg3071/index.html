Yes, that's a pothole. I just made a bunch of circles inside themselves to make a "bowl". If the map is too big then the pothole will not work, so keep the model small enough that the pothole works.


Some other clarification:


I made this for CarbonFest, so I put in a viewing area for spectators over by where it says Carbon Combat. You swerve into the other lane and back again to avoid it, doubling it as an obstacle.


This was actually made on a tileset of highway pieces which were loaded in then put into the model. For instance, there are four types of curves, each type being the same to all others of it's type. You can see the "grid" pattern with each tile piece having a support pillar in it's center (with the exception of the curves).


I epicly failed on the turns. I basically drew a circle, then cut it into fourths and used that as a turn. But it went wrong somehow, glitching/reversing faces on each turn plus making the yellow median go off-center in some areas.


The model's in public domain right now. If you can fix it, it's yours...but it would be nice if someone posted a fixed version here.
