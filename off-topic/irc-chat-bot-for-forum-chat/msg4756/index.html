IRC Chat Bot for Forum Chat
posted Early January, 2013
EDIT:
My apologies, I realize that I made a slight mistake in my earlier message: Due to the fact that CarbonTech is not the administrator of TinyChat, in order to upload an IRC Chat Bot to the Forum Chat, we would have to create a new forum chat using an IRC system. It would not be too hard to create it, but embedding it may present challenges. I will see what I can do about this.


ORIGINAL POST:

Hello,


Recently, there has been a complaint about people moderating the chat room on the forum ineffectively, or simply not moderating at all. I decided that a simple solution to this would be to create an IRC chat bot, upload it to the channel that the Forum Chat room is on, and allow it to do the moderating, rather than having an actual person do it. This one main drawback to this method is the fact that it would have to be placed on a server with high uptime- if the server crashes, the IRC Chat Bot does not work during server downtime.


An IRC Chat Bot could provide multiple functions to the Forum Chat:
Commands: Commands such as /ban, /dateandtime, /evict, /members, etc. All of these commands would be relayed to the chat bot, the bot would interpret these commands, and would perform an intended function. Commands could vary based on the need for different functions.
Relaying Chat Messages: The IRC Chat Bot could relay messages with certain words or phrases to the PM of a moderator, so that they could deal with the problem appropriately. As well, it could also ban users with a matching IP of users who used certain words and phrases that were insulting or explicit.
Moderator Reporting System: Rather than sending screenshots of something someone said in a post, why not have the IRC bot send it automatically? By using a command, such as /sendtomod, it would record the conversation, and would send the information to a moderator, along with recording the IP addresses of people involved (IP address recording is optional).


Please, feel free to present ideas and opinions on this idea- if reaction seems positive, I will create this IRC bot when I next have a little bit of free time.


Sincerely,
