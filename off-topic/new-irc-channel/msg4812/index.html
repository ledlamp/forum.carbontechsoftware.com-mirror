New IRC Channel
posted Mid-January, 2013
Hello,


Recently, I created a new IRC channel for this community. It is hosted through Mibbit, an online IRC hosting site. To access this new IRC chat, follow these instructions:
1) Go to http://client01.chat.mibbit.com/
2) To connect to a server, select the option of "Mibbit [webirc]" from the dropdown menu.
3) For a nickname, put in any nickname you choose (preferably your forum name so people can identify with you.
4) The channel for this IRC chat is "#Carbon_Combat" (don't type the quotes).
5) After all of that information is inserted into the boxes, click "Connect", and it will connect to the new IRC chat room.


The main advantage of a separate IRC is that I will be able to upload an IRC bot into the chat room, if it is needed at any point.


Please PM me if there are any problems connecting, or simply state your concerns in the comments section.
