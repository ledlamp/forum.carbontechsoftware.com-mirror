Announcements / New Anti-Spam Defenses
« on: Early July, 2012 »
The forum got its first spammer a few hours ago - which is not all that surprising I suppose. I took the liberty of putting up a few barriers that should make it considerably harder for future spammers (both humans and bots) from attacking the forum.

The first line of defense is the new registration requirements. There are now a plethora of questions (all fairly simple) that should deter all bots and most human spammers. I won't discuss the questions or the answers here due to security concerns.

The second line of defense is Akismet. Akismet automatically flags and hides any threads that contain spammy content. If any spammers do manage to get through the registration system they will have to be incredibly clever in order to sneak past Akismet.

The third line of defense is the moderators. There are currently four moderators on this forum which should be more than enough to clean up anything that gets through Akismet.

The last line of defense is of course you. If you see anything that is blatantly spam please report it using the handy link at the bottom of each post/comment.

Hopefully all of this will make for a nice and clean forum.

Cheers,
CarbonTech

Announcements / Carbon Combat 2.9 Alpha
« on: Early July, 2012 »
Greetings!
 
This version contains a new game mode and many tweaks and bug fixes to the overal gameplay. It also marks a new era for the forum: a transition away from Plexpedia to SMF (Simple Machines Forum). As usual, below is a brief description of the changes made.
 
Survival Gameplay Mode
This gameplay mode aims to level the playing field for newbies as well as introduce a new challenge for veterans. All players spawn with their equipped vehicle but with no armor and only basic weapons (rocket launcher, plasma blaster). Weapons and armor can be picked up around the map where health pickup items are usually spawned (displayed as crates) as well as from the usual weapon pickup locations. Thus, having prior knowledge of where health and weapon pickup locations are is crucial in this gameplay mode. As there are is no way to regain health (ninja regen armor is not available), making sure you don't lose health from unnecessary confrontations is also key to surviving.
 
Item Drop Rates
Crates spawn every 150 seconds and have a 50/50 chance of distributing a weapon or armor. Weapon distribution is completely random but armor is not:
 
30% chance for basic
30% chance for light
20% chance for assault
10% chance for heavy
10% chance for ninja invisibility
 
Please note that there may be bugs (like always) with this gameplay mode even though it has undergone rigorous testing. Special thanks to ibcf and nattyfrank for helping to alpha test.
 
What to Expect
Less bugs, the ability to pick up other players' dropped armor/weapons and more (if you suggest it.


 
New Forum System
After much consideration I have decided that in order for the game to continue to expand, a more robust forum system is needed. This new forum system (powered by SMF) provides just that. Below is a quick FAQ regarding the switch:
 
Can I log into the new forum using my forum/game account?
Unfortunately not. Please register your usual username with the new forum system.
 
Will my old posts be transferred to the new forum?
No - Plexpedia does not have an export option. You are welcome to repost the threads you posted on the old forum system.
 
Why is the url forum2 instead of forum?
It will be changed in a few days after everyone has adjusted their bookmarks/seen this announcement.
 
Who hosts this forum?
I do. The forum is hosted on the same webserver that the game store is. As such, please be aware that I am paying for the bandwidth.
 
> Link to the new forum



Optimizations
I further optimized the game which should make it run even faster when there are lots of players on-screen. Ranking lookups were also optimized and should use less CPU as well as return the data quicker.
 
Standalone
Check back here tomorrow afternoon

Other Changes/Bug Fixes
  • Racer accel increased.
  • Grenade launcher, EMP grenade launcher and the laser cannon now show a charge bar. Charging more for the launchers launches the projectile farther.
  • Guests can now enter their own name (with guest_ prepended).
  • Game mode descriptions are now shown on entering servers.
  • Selecting text in the chat box will no longer cleared enter text.
  • Friend joining should work now.
  • Health and invisibility pickups should respawn each round.
  • AI server amounts decreased.
  • End of round defeat message hidden.
  • Wheels not turning invisible for the default car fixed.
  • Laser cannon should be able to blow up oil barrels now.
Known Bugs
If you know any others, please post them here or send them to me via PM or email.
  • Health pickups in non Survival game modes not showing up.
  • Player cap amounts not resetting when switching between maps.
