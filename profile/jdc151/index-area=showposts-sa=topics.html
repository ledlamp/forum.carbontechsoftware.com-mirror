I have noticed in past topics and in game that there are many problems with using a gas flamethrower. If CT utilized a liquid flamethrower, a lot of these problems would be rectified. Feedback & Ideas / New game modes?
« on: Late August, 2013 »
I've looked it up, and apparently only a few have suggested new game modes. As of right now, there are only three...deathmatch, ctf, and survival. People have been asking especially for a racing mode, where only racing maps can be put on, and have limited flying time, with the ability to respawn at a certain point on the track. Another useful option for future game mods would be to judge fan made maps, for instance, the secret base, castle 2, among others.
There could be an appointment when all global moderators try out one map, record it, and play around in it adding custom map items, like launching ramps, weapon caches, etc. which reminds me of another cool game mode, tounaments. Although this has been already implemented, it could use some upgrades, such as the ability to have a custom map with either rounds or heats, with only one weapon able to be equipped. It would be very interesting if any of these were taken into consideration..
