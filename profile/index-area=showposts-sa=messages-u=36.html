« on: Mid-April, 2013 »
Hello all,


Recently, I have been reading about this "hacker" problem that we are having, and while there are many suspicions about some players being hackers, the actual evidence to support these hypotheses are weak. For this reason, over the past few days I have been developing some code to allow access to the database of usernames and IP addresses currently stored in Carbon Combat.


A quick note about personal security: this code will not be able to be used by anybody, it will only be implemented on admin and moderator accounts.


A few quick notes about this code:
  • It begins by creating a class object that replicates the mapping of usernames to IP addresses. The code allows users to add names and IP addresses, delete names and IP addresses, and (the new function) use a function that, upon implementation, intakes a name of a player, and returns the IP address.
  • This code will NOT work in Carbon Combat, as currently it is designed to run only on a local machine. It will not work in the actual game; it will only retrieve IP addresses based upon those already added to the database that maps usernames to IP addresses.
This code is written in Python 2.7.3, and is saved as a .py file. Please note that when downloading Python, it is recommended to download Python 2.7.3 over Python 3.3.1 (the latest version) due to some incompatibilities between the two versions. My code can be both viewed and downloaded at the following link:


DOWNLOAD THE CODE HERE


RECOMMENDED SOFTWARE TO OPEN THE PROGRAM:
It is recommended that users have the Python package installed on their computer. This package includes the basic Python framework, the IDLE interactive shell, and some useful Python modules. Unless the user has some other Python software installed on their computer, they will most likely be unable to open .py files. This software can be downloaded from the Official Python Webpage, at http://www.python.org/download/.


MORE RECOMMENDED SOFTWARE:
More software that is recommended instead of the Python package is Python from the Enthought Python Distribution Package. This package includes the Python framework, the IDLE interactive shell, as well as far more modules than the basic Python framework. This may prove useful when running code that contains a module that is not pre-installed in the basic Python package. This software can be downloaded at the Enthought Website, at https://www.enthought.com/downloads/.


Both packages run on multiple operating systems- it is likely that the program will be able to run on your computer.


HOW TO USE THE PROGRAM:
Use of the program is quite simple; the instructions here are just to get you started.
NOTE: Any text inside of two backslashes '//' means that this is not EXACTLY what to type into the program. E.g., saying type '//your username' means that you type in your Carbon Combat username, not "your username".
1) Open your program in IDLE or another Python editor
2) Run the program
3) Type in:
'//something' = storeIP()
4) Type in:
'//same something'.addName('//name', '//IP address')
* Continue doing this for as many names as you would like to add to your local database of names and IP Addresses.
5) Type in:
'//same something'.getIP('//name')


This will either:
  • Raise an error (if the name is not in the database of known names and IP addresses
  • Return the IP Address associated with the inputted name
