Carbon Combat / My apologies
« on: Late October, 2013 »
I formally apologize to everyone for my long absence from carbon combat. Due to the stagnant development of the game, I had only been checking the forums now and then and playing for a few minutes when I felt like it. As I was browsing the forums today I realized that CC is on the verge of a major change or an eternal standstill. Many new players have joined up too. One of the main reasons I left was due to shortage of players. However the game seems to have acquired new players much to my relief. I really admire cremy, speed and lamp for sticking with CC continuously for such a long time. Due to the mysterious absence of CT (with some allegations that he is dead) no new feature have been added to the game. So I came up with a rather ambitious idea.
Why don't we try improvising new features within the current system? I mean that we could make up ideas for new features that don't require changing the code of the game instead of relentlessly posting new ideas on the forums about new weapons and bodykits etc. that would not be implemented in a long time even if CT were here. For a start, the clan system was a brilliant idea but not was not properly implemented. In fact I think it was not implemented at all. In order to implement this properly we must first bring about other changes. For example, there should only be two clans (more like factions) and all active players should play on the same server. A good way to make this possible is by
making a list of all active players and if any server is being hosted by an active player all players should join that particular server. We could also make a log on the off-topic forums about win-loss count of the aforementioned clans. Please comment on what you feel and again, I apologize for my absence.
Also forgive me if this is topic is not in the proper category of the forum.

PS.: Anyone still remember whose idea the clan system was? I just wanna know. Was it speed or lamp or cremy? Can someone tell me?
