Please note: There is a BIG chat bug.

The in-game chat dialog reacts to in-game control buttons...and Vice-Versa.

For example, if you are using the "in-game chat" and press the "p" key while typing a word in the chat window- you will inadvertently pause the game.Additionally, pressing directional control keys (W/A/S/D) while chatting or pressing the SPACE bar, will effect the vehicle. The problem gets exacerbated by the fact that the in-chat window gets flooded by random repeat letters (like directional keys) and other multiple key presses from players trying to move their vehicle, but can't; due to the system registering the key presses as a in-chat text.

This makes chatting with players in-game a veritable nightmare. (._.)

Nota Bene: Sorry, CT. I love your code; but this bug is bugging me.
