2.9.2 Alpha Release
posted Late December, 2012
This release that adds GTR's Devastator, a new car paint shader, some weapon balances, and lots of bug fixes.





Changelog
+ GTR's Devastator muscle car (high acceleration, medium-high top speed, low handling)
+ New shaders for the car body and windshield
• Fixed the profile not loading
• Fixed Creator plateaus turning into craters bug
• Fixed Creator health pickups
• Fixed large Creator worlds causing Photon to lose connection on load
• Fixed soldier crouching bug
• Fixed melee slicer blade damage not sending
• Fixed laser cannon being able to kill self
• Fixed music loading
• Fixed preloader click ignore
• Fixed minor connection issues greying out the server list
* Increased melee slicer blade collision box
* Decreased shotgun damage
* Increased railgun reload time
* Increased flip trigger box size
- Disabled my Dropbox hosted worlds from appearing in the more maps tab
