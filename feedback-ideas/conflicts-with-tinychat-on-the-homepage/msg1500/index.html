Conflicts with TinyChat on the homepage
posted Late August, 2012
So, the chat is on the homepage. The problem with it being there is:
  • Increases load time of the homepage, causing lag for some people.
  • Makes items not clickable until chat is loaded (happens for me, probably not for everyone)
  • Makes unnecessary guests come and go.
There is a temporary solution until (if it does) get removed from the homepage. You can install the FlashBlock add-on for Chrome or Firefox which will prevent it from loading. If you want it to load, just click it.
