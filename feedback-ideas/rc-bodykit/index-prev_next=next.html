Convenient image uploader
posted Mid-December, 2012
It would be nice if this forum had an image uploader like ATI/Plexpedia forums do (old forum was powered by Plexpedia). It would make it so much easier to upload an image or images.


With the imgur uploader (or any uploader), I have to open a new window or tab, upload the image(s), copy the codes, and paste it into the post. With Aubrey's uploader, you just put your cursor (blinking line) where you want to place the image, click the tree, upload an image, and insert it! No need to make a new window or tab which I have to switch to a lot. Especially with a window: it always hides behind my window with the posting page.


Perhaps CT could insert/make an image hosting tool that does not require any copying & pasting and switching tabs/windows.


Note: don't use Postimgae.org image hosting, that one sucks.




It should be powered by something like imgur. Aubrey's uploader hosts images on his server, but it does not have to be on this site. I've seen logo making sites that use imgur to host stuff.

1. Click the tree in the toolbar


2. Click Choose File and select an image



3. Click upload! Enter a description (optional) and/or change the style (optional).


The image is now in the post!
