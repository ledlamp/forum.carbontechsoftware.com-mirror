Hello,

According to the forums rules (link in my signature) you are allowed one forum account.  Nothing is said about game accounts. 

Rule 4: Each person is restricted to possessing one forum account. Send a PM to a moderator or administrator if you need an exception (e.g. switching accounts).

I personally have two accounts, but that is a long story.  If you want to create a new account go to one of the sites that CC is hosted on, like Kongregate.com, and register a new kongergate account then visit the game on that site.  You will now have a new account. 

If you do make a new account, they must not interact in any way though.
